Provides a decision tree used to correctly indent the current line.

The tree is composed of {@link IndentRule}s, and asks a series of yes or no questions about the current position before determining the appropriate action. All nodes use helper methods from {@link DefinitionsDocument} and the Reduced Model package for context.

The tree itself is built in the {@link Indenter} class, where a singleton instance is made available to DefinitionsDocument for using on a single line. An outline of the tree is given below.

  • 1. QuestionInsideComment
    • 2. Yes: QuestionPrevLineStartsComment
      • 3. Yes: QuestionCurrLineEmpty
        • 4. Yes: ActionStartPrevLinePlus(" * ")
        • 5. No: ActionStartPrevLinePlus(" ")
      • 6. No: QuestionPrevLineStartsWith("*")
        • 7. Yes: QuestionCurrLineStartsWith("*")
          • 8. Yes: ActionStartPrevLinePlus("") (Goto 12)
          • 9. No: QuestionCurrLineEmpty
            • 10. Yes: ActionStartPrevLinePlus("* ")
            • 11. No: ActionStartPrevLinePlus("") (Goto 12)
        • 12. No: ActionStartPrevLinePlus("")
    • 13. No: QuestionBraceIsParenOrBracket
      • 14. Yes: QuestionNewParenPhrase
        • 15. Yes: ActionBracePlus(" ")
        • 16. No: ActionBracePlus(" " + 1 level)
      • 17. No: QuestionBraceIsCurly
        • 18. Yes: QuestionCurrLineStartsWithSkipComments("}")
          • 19. Yes: ActionStartStmtOfBracePlus("")
          • 20. No: QuestionStartAfterOpenBrace
            • 21. Yes: ActionStartStmtOfBracePlus(1 level) (Goto 34)
            • 22. No: Goto 23.
        • 23. No: QuestionStartingNewStmt

            • 25. Yes: QuestionExistsCharInStmt("?", ":")

              • 27. No: ActionStartStmtOfBracePlus(1 level) (Goto 34)
            • 28. No: ActionStartPrevStmtPlus("")

              • 32. Yes: QuestionExistsCharInStmt("?", ":")

                • 34. No: ActionStartStmtOfBracePlus(1 level)
              • 35. No: ActionStartCurrStmtPlus(1 level)
